# FreelanceFlow – Smart Freelance Project & Income Tracker

FreelanceFlow is a MERN stack web application designed to help freelancers manage projects, monitor income, and organize client work efficiently.

## 🚀 Features

- User Authentication
- Project Management
- Income Tracking
- Dashboard Overview
- Client Project Monitoring
- Responsive UI

## 🛠 Tech Stack

Frontend:
- React.js
- CSS

Backend:
- Node.js
- Express.js

Database:
- MongoDB

## 📁 Project Structure

FreelanceFlow/
│
├── frontend/
├── backend/

## ⚙️ Installation

### Clone Repository

git clone <repository-url>

### Frontend Setup

cd frontend
npm install
npm start

### Backend Setup

cd backend
npm install
node server.js
